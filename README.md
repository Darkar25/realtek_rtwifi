  ### Driver supports
  ```
  * WPA3-SAE
  * MESH (HT-IBSS) mode
  * AP mode
  * P2P mode
  * Monitor mode
  * Frame injection (packet injection)
  * Virtual interfaces (VIF)
  * Newest mac80211
  * Channel width
```



  Please report issues (if you encounter any), open a issue report with as much details/logs
  as possible.
